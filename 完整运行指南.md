## 常见问题排查

### 问题 A：`ModuleNotFoundError: No module named 'xxx'

**解决方案**：
```powershell
# 重新安装依赖
py -3.9 -m pip install -r requirements.txt

# 如果某个特定包缺失，单独安装
py -3.9 -m pip install 包名
```

### 问题 B：端口被占用（Port already in use）

**解决方案**：
```powershell
# 查找占用端口的进程
netstat -ano | Select-String -Pattern ":8001"

# 终止进程（替换 <PID> 为实际的进程 ID）
taskkill /F /PID <PID>
```

### 问题 C：数据库连接失败

**检查项**：
1. MySQL 服务是否启动？
2. 用户名密码是否正确？
3. 数据库 `travel_rag` 是否存在？

### 问题 D：API 调用失败

**检查项**：
1. 网络连接是否正常？
2. API Key 是否有效？
3. API URL 是否正确？
